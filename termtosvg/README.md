# termtosvg
>you can copy the text from svg file.

**start termtosvg rec**
>`python3 -m termtosvg`

**start termtosvg with play pause button**
>`python3 -m termtosvg -t window_frame_js`

**stop termtosvg rec**
>`exit` or `Ctrl+D`
